# Pocket Host 产品概述

## 产品定位
Pocket Host 是一个移动端点对点游戏平台，允许设备在**无路由器、无互联网**的环境下建立局域网连接进行多人游戏。

## 核心理念
- **App 是"主机"，游戏是"卡带"**：类似游戏机的设计理念
- **任意设备做主机**：Android 或 iOS 设备均可作为 Host
- **扫码连接**：通过二维码实现快速、可靠的设备配对

## 技术亮点
- **网关反向推导**：绕过 iOS 屏蔽广播的限制，Client 连接热点后直接向默认网关 IP 发起单播握手
- **混合权威架构**：Host 只计算核心逻辑（裁判），Client 负责渲染和预测，降低手机作为服务器的发热
- **动态游戏分发**：通过局域网 CDN 分发 `.pck` 游戏资源包，无需更新 App 即可获取新游戏

## 开发阶段
1. **Phase 1 - 连接原型**：验证 iOS/Android 双向 TCP/UDP 通讯（当前阶段，iOS 优先）
2. **Phase 2 - 游戏化 MVP**：实现一个多人实时互动游戏
3. **Phase 3 - 平台化**：实现动态加载游戏卡带
4. **Phase 4 - 生态化**：开放 SDK，允许第三方开发游戏
